## 📌 프로젝트 개요

### 1️⃣ 주제 및 선정 배경
본 프로젝트는 **1인 미용샵을 위한 간편한 예약 관리 시스템**입니다.
기존의 타 업종까지 섞여있는 예약 관리 방식에서 1인 미용샵을 선호하는 고객을 위한 미용샵 집중 예약 관리 방식으로 서비스 개선하였습니다.

또한, JSP와 Servlet 기반으로 웹 개발의 핵심인 CRUD 중심으로 다뤘으며, 
MVC 패턴을 적용하여 실무 역량 강화에 집중하였습니다.

< 아키텍처 구조 >
- Service
    - 비즈니스 로직
- DAO (Servlet)
    - DB 접근 및 데이터 처리
- Servlet (Controller)
    - 요정 및 응답 처리, 흐름 제어
- JSP (View)
    - 사용자 화면 렌더링

### 2️⃣ 기능 소개
- 회원 : 회원가입, 로그인/로그아웃, 마이페이지
- 미용샵 : 회원가입, 로그인/로그아웃, 마이페이지
- 예약 : 매장 선택 및 스케줄 예약, 예약 조회 (user/designer)
- 마이페이지
    - 회원용 : 회원 정보 수정, 리뷰 관리, 예약 조회
    - 매장용 : 회원 정보 수정, 예약 리스트 및 상세 조회, 리뷰 조회
- CS 문의게시판 : 매장별 문의 게시판 등록, 수정, 삭제

👥 팀 구성 및 역할
- 안영아
    - 발표 자료 : 구조도 설계, 포트폴리오
    - 레이아웃 : 마이페이지 고객용/매장용
    - 기능 구현
        - 로그인 및 로그아웃 (고객/매장)
        - 회원가입 (고객/매장)
        - 회원정보 조회 및 수정
- 조성진
    - 발표 자료 : 요구사항 및 기능 정의서
    - 레이아웃 : 회원가입 (고객/매장), 매장별 리뷰
    - 기능 구현
        - 매장별 리뷰
        - 고객 리뷰 등록 및 수정
        - 매장 이용한 고객 리뷰 조회
- 김현수
    - 발표 자료 : 포트폴리오
    - 레이아웃 : 로그인 (고객/매장), 매장 예약
    - 기능 구현
        - 로그인 및 로그아웃 (저장/찾기)
        - 매장 예약 및 조회
        - 예약된 고객 조회 및 예약 변경
- 유지상
    - 발표 자료 : ERD, 테이블 정의서
    - 레이아웃 : 메인 화면, C/S 문의게시판
    - 기능 구현
        - C/S 문의 게시판 매장별 조회
        - 매장별 문의글 고객 작성 및 수정
        - 고객 문의글에 대한 매장 답변 작성 및 수정
